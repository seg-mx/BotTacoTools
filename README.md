# BotTacoTools

---

This is a version of TacoShack implemented in a Discord Bot

## Setup

---

Add a `.env` file with the following:

```env
OWNER_ID=<your_id>
CHANNEL_ID=<channel_for_cooldown_mentions>
TOKEN=<bots_token>
```

## Dependencies

---

This bot uses the following:
* discord.js v13.7.0
* dotenv v16.0.1
